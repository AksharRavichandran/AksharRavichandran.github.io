# AksharRavichandran.github.io

Personal website built with [React](https://react.dev/) and [Vite](https://vitejs.dev/).

## Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Run the linter:

```bash
npm run lint
```

Build for production:

```bash
npm run build
```
